# Titanic Mini Visualization Dashboard

Dataset Used:
Titanic Dataset

Tools:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

Visualizations:
1. Age Distribution
2. Survival Rate by Gender
3. Fare Distribution by Passenger Class
4. Age vs Fare Scatter Plot
5. Correlation Heatmap
6. Survival Rate by Age Group

Key Findings:
- Female passengers had higher survival rates.
- First-class passengers had better survival chances.
- Higher fare passengers showed improved survival probability.
- Children had relatively higher survival rates.

How to Run:
Open the notebook in Google Colab and run all cells.
